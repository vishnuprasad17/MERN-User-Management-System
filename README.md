# MERN User Management System

## Overview

MERN User Management System is a full-stack web application that facilitates user registration, authentication, and profile management. It incorporates the MERN stack (MongoDB, Express.js, React, Node.js), Redux for state management, and Cloudinary for image uploads.

## Features

- **User Authentication:** Secure user registration and login using JWT tokens.
- **User Profiles:** View and update user profiles, including profile pictures uploaded to Cloudinary.
- **Admin Panel:** Admin functionalities for searching, deleting, blocking, updating, and unblocking users.
- **Middleware:** Express-async-handler for asynchronous route handling.
- **Redux State Management:** Centralized state management for enhanced scalability.
